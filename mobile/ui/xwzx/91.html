8个巧妙的UI设计：给你带来麻烦，却让你赞不绝口

日期：2022-01-11 11:48:27 来源：舜王科技
每逢节日，你是否注意到了大家在使用的技术？你可能会注意到，不同年代出生的人都在把弄着触摸屏。

这太疯狂了！但这就是新常态。随着触摸屏的崛起，计算机不那么让人感到困惑了。但这并不意味着轻触、滑动和捏放就是终点。在 2014 年，移动软件试图在简洁性和功能强大方面取得平衡。我们也看到了让手机与其他机器更和谐共处的努力。而围绕这一方面，人们也越发意识到，触摸屏并不总是答案。

下面，我将向大家呈现 2014 年出现的一些最佳 UI 创意。

Lollipop 的锁屏通知
Material Design 是谷歌在交互性上制定的新规则。这一新系统的首批产品之一就是最新版安卓操作系统 Lollipop。而锁屏通知则是 Lollipop 最好的地方之一。

8个巧妙的UI设计：给你带来麻烦，却让你赞不绝口

锁屏通知让锁屏的功能更加强大。用户可以滑动通知来删除，也可以捏放来打开，比如可以在通知栏查看更多邮件内容，而无需进入收件箱。另外，Lollipop 让用户可以很方便的对通知进行设置，从而允许一些通知显示，同时屏蔽其他通知。这样，用户可以轻松的关闭其他通知，只保留电子邮件通知，却无需深入到设置中关闭。

总之，谷歌试图让用户尽可能在系统表层完成任务，而不用深入系统内部。

无处不可发短信
这个功能不是很耀眼，但过去一两年中也没有其他功能像可以从电脑上发短信一样，让我们的生活变得更轻松。在过去几版 OS X 操作系统中，苹果一直都在整合消息服务，并在 Yosemite 中迈出了最后一步：让用户不仅可以在电脑上发送 iMessage 消息，还可以发送短信。

8个巧妙的UI设计：给你带来麻烦，却让你赞不绝口

这个功能很好，文本消息（包括 iMessage 和短信）仍然是我们最常用的沟通手段，但常用的设备仍然是笔记本电脑。如果可以从电脑上发送短信将节省大量时间。虽然现在这一功能还不是百分百完美，比如有时候会听到手机在其他房间中响，但等上几分钟后消息才会显示在 Mac 电脑上，不过它工作得已经很好了，尤其是和早期 iMessage 相比时，当时的 iMessage 经常没法同步。

无可否认的是，短信无处不在会导致一些社交怪癖，在电脑上快速发送短信，就像在使用即时通讯工具一样。如果消息堆积如山时，收到消息的人却还在笨手笨脚地用触摸屏键盘打字，这是怎样的一幅情景啊！

为汽车开发的智能触摸屏
当人们把汽车无线电或空调的控制权交给一堆触摸屏按钮时，就迫使司机们必须把视线从道路上移开。这简直是太愚蠢了，然而汽车制造商们依然在这么做。好吧，如果它们坚持要使用触摸屏，也至少思考一下让人们不看触摸屏就能使用的方法吧。

设计师 Matthaeus Krenn 深思熟虑的概念设计就做到了这一点。他的设计并不依赖于小的虚拟按钮，而是将功能赋予了更宽泛的手势。在触摸屏任何地方用两根手指滑动都可以调节音量，四根手指滑动则可以控制温度。的确，用户需要学习怎么来操作，而且最终都会学会，因为基于按钮的触摸屏总免不了要司机看上一眼。
